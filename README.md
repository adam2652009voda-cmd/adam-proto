# Adam Abdalla Elseby — Portfolio

A modern, premium personal portfolio website for **Adam Abdalla Elseby**, Creative Editor & Web Designer.

## Features

- **Dark/Light Mode** — Toggle with persistent preference via localStorage
- **Fully Responsive** — Mobile-first design optimized for all screen sizes
- **Glassmorphism UI** — Premium frosted-glass card effects
- **Smooth Animations** — Scroll reveals, hover effects, animated counters
- **Project Filtering** — Filter portfolio by category (Video, Web, Branding, Graphic)
- **Testimonials Carousel** — Auto-playing slider with touch/swipe support
- **Contact Form** — Validated form with success feedback
- **SEO Optimized** — Semantic HTML, meta tags, and accessible markup

## Sections

1. Hero — Introduction with profile image and CTAs
2. About Me — Professional biography
3. Core Skills — Primary and secondary skill cards
4. Services — Professional service offerings
5. Portfolio Projects — Filterable project showcase
6. Experience & Achievements — Animated stat counters
7. Testimonials — Client review carousel
8. Contact — Form and social media links
9. Footer — Quick links and copyright

## Getting Started

Open `index.html` in a browser, or serve locally:

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .
```

Then visit `http://localhost:8080`

## Customization

- **Profile photo**: Replace the placeholder in the hero section with an `<img>` tag pointing to your image in `assets/`
- **Social links**: Update `href="#"` values in the contact and footer sections
- **Projects**: Edit project cards in `index.html` and add real images to `assets/projects/`
- **Contact email/phone**: Update in the contact section
- **Colors**: Modify CSS variables in `:root` at the top of `css/styles.css`

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (No dependencies)
